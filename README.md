## 📖 Tentang

**Learnix Space** adalah platform belajar berbasis website untuk mahasiswa dan *fresh graduate* di bidang **Teknologi Pendidikan (TP)**. Platform ini menyediakan materi pembelajaran, referensi proyek, dan showcase karya untuk mengasah skill yang relevan dengan kebutuhan industri dan mempersiapkan diri menghadapi dunia kerja.

## Fitur

- **Learning Experience Design** — pengalaman belajar yang efektif, interaktif, dan berpusat pada pengguna
- **Industry-Relevant Projects** — kumpulan proyek yang mencerminkan kebutuhan nyata dunia kerja (E-Learning, UI/UX, Digital Campaign, Training Program, Virtual Experience, LMS, Mobile Learning)
- **Digital Learning Solutions** — solusi pembelajaran berbasis teknologi
- **Galeri & Alumni** — showcase karya dan profil alumni berprestasi
- **Responsive Design** — tampilan menyesuaikan desktop maupun mobile
- **Galeri Popup Interaktif** — preview gambar tanpa pindah halaman

## Halaman

| Halaman | Deskripsi |
|---|---|
| `index.html` | Landing page — hero, keunggulan, galeri, alumni |
| `project.html` | Daftar proyek + halaman detail per proyek |
| `about.html` | Profil, misi, dan layanan platform |

## Teknologi

- **HTML5**
- **CSS3**
- **JavaScript** (vanilla)
- **Deployment:** Vercel

## Menjalankan Secara Lokal

```bash
# Clone repository
git clone https://github.com/RunayAnakElang/learnixspace.git

# Masuk ke folder proyek
cd learnixspace

# Buka index.html langsung di browser,
```
