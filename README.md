# Learnix Space

**Learnix Space** adalah platform belajar berbasis website untuk mahasiswa dan 
*fresh graduate* di bidang **Teknologi Pendidikan (TP)**. Platform ini menyediakan 
materi pembelajaran, video tutorial, dan referensi proyek untuk mengasah skill yang 
relevan dengan kebutuhan industri dan mempersiapkan diri menghadapi dunia kerja.

🔗 **Live Demo:** [learnixspace.vercel.app](https://learnixspace.vercel.app)

## Fitur

- **Learning Experience Design** — pengalaman belajar yang efektif, interaktif, dan berpusat pada pengguna
- **Industry-Relevant Projects** — 7 contoh proyek yang mencerminkan kebutuhan nyata dunia kerja (E-Learning, UI/UX, Digital Campaign, Training Program, Virtual Experience, LMS, Mobile Learning)
- **Digital Learning Solutions** — solusi pembelajaran berbasis teknologi
- **Galeri & Alumni** — showcase karya dan profil alumni berprestasi
- Desain responsif dengan galeri popup interaktif

## Halaman

- `index.html` — Landing page (hero, keunggulan, galeri, alumni)
- `project.html` — Daftar proyek + halaman detail per proyek
- `about.html` — Profil, misi, dan layanan platform

## Teknologi

- HTML5
- CSS3
- JavaScript (vanilla)
- Deploy: Vercel
